## 📋 О проекте

Проект представляет собой полноценную мехатронную систему управления температурным режимом лабораторного термостата. Система разработана на базе гетерогенной архитектуры National Instruments и предназначена для использования в образовательных целях (Кванториум, вузы) для изучения принципов автоматического регулирования, работы с АЦП/ЦАП и промышленными протоколами.

### 🎯 Ключевые возможности
*   **Точное поддержание температуры:** Использование П-регулятора с ШИМ-модуляцией для плавного управления мощностью нагрева.
*   **Двухконтурное управление:** Автоматическое переключение между нагревом (лампа накаливания) и охлаждением (вентилятор).
*   **HMI-интерфейс:** Удобная сенсорная панель NI TPC-2206 с визуализацией переходных процессов в реальном времени.
*   **Надежность:** Детерминированное выполнение кода благодаря ОС реального времени (RTOS) на контроллере NI cFP-2220.
*   **Безопасность:** Встроенная система обработки ошибок, аварийное отключение и защита от перегрева.

## 🏗 Архитектура системы

Система построена по клиент-серверному принципу:
1.  **Серверная часть (Controller):** Выполняет сбор данных с термопары, расчет управляющего воздействия (ШИМ) и дискретное управление реле. Работает в жестком реальном времени.
2.  **Клиентская часть (HMI):** Отвечает за отображение телеметрии, ввод уставок пользователем и построение графиков.
3.  **Коммуникация:** Обмен данными осуществляется через **Network Shared Variables** по протоколу Ethernet, что обеспечивает высокую скорость синхронизации без написания низкоуровневого сетевого кода.

## 🚀 Быстрый старт

### Требования к оборудованию
*   Контроллер NI cFP-2220 с шасси Compact FieldPoint.
*   Панель NI TPC-2206 или ПК с установленным LabVIEW.
*   Модули: cFP-TC-120, cFP-PWM-520, cFP-RLY-425.
*   Лабораторный стенд "Термостат".

### Установка и настройка
1.  Подключите все модули к шасси контроллера согласно схеме в **Приложении Б**.
2.  Запустите утилиту **NI MAX** на панели оператора.
3.  Убедитесь, что контроллер обнаружен в сети и имеет корректный IP-адрес.
4.  Откройте проект LabVIEW (`Thermostat_Control.lvproj`).
5.  Разверните приложение на контроллере (Deploy to Target).
6.  Запустите лицевую панель на TPC-2206.
